# GoTTH - Docker 
idea and initial structure from: https://github.com/tomdoesTech/gotth

* Go - Backend
* Tailwind - CSS
* Templ - Templating
* HTMX - Interactivity

# Run Application 

build container: 
```
docker build --tag {container-name} . 
```
run container:
```
docker run -p 8080:8080 {container-name}
```
